# いわら保育園 Tシャツ注文集計システム

Tシャツトリニティの商品を参考にしながら、園内で職員が **デザイン / 色 / サイズ** を選んで送信できるフロントエンドと、
その内容を集計できるバックエンドをまとめたシンプルなフルスタックアプリです。

## できること

### 職員向けページ (`/`)
- 公開中のTシャツデザインをカード表示
- デザイン / 色 / サイズを選択して注文送信
- 備考欄に「2枚目希望」「サイズ相談」などを書ける
- 送信内容は管理画面で確認可能

### 管理画面 (`/admin`)
- デザイン登録 / 編集 / 公開ON-OFF
- デザイン画像URLの登録、または画像アップロード
- 色候補、サイズ候補の設定
- 注文一覧の確認
- 発注用集計（デザイン × 色 × サイズごとの合計）
- 備考欄だけを抜き出して確認
- CSV出力 (`/api/export/orders.csv`)

## セットアップ

```bash
cd /mnt/data/tshirt-order-system
npm install
npm start
```

起動後:
- 注文ページ: `http://localhost:3000/`
- 管理画面: `http://localhost:3000/admin`

## データ設計

### designs テーブル
- `name`
- `source_url`
- `description`
- `design_image_url`
- `colors_json`
- `sizes_json`
- `active`
- `display_order`

### orders テーブル
- `staff_name`
- `department`
- `design_id`
- `design_name`
- `color`
- `size`
- `quantity` (固定1)
- `note`

## 補足
- このアプリは **Tシャツトリニティ上で自動注文する仕組みではありません**。
- 最終発注は、管理者が集計画面を見ながら人力で行う前提です。
- 自動スクレイピングは初期実装に入れていません。運用を安定させるため、管理画面から手動で最終確認・修正する方針です。
- 色ごとの可変プレビューは、デザイン画像（透過PNG推奨）を登録すると見た目がきれいになります。
